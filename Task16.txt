1] Что такое тэг и из чего он состоит?

ТЭГ – это ключевое слово, по которому без проблем можно найти требуемый материал. ТЭГ состоит из Ключа (Key) и Значения (Value)

2] Что такое атрибут?

Атрибуты HTML - это специальные слова, используемые внутри открывающего тега для управления поведением элемента. Атрибуты HTML являются модификатором типа элемента HTML.

3] Сколько атрибутов может быть в одном тэге?

В одном Тэге может быть один, два и более атрибутов.

4] Как искать элемент на странице?

Нажать Inspect и выбрать нужный элемент на странице.

5] Какими способами подключается css?

CSS - это набор стилей.

Способы подключения:

1. Написать Тэг style

2. Вставить стиль как атрибут у какого-то Тєга

3. Через Тэг Link
