
API запрос состоит:

- Request URL  - это адрес, по которому мы посылаем запрос.

- Заголовки - позволяют клиенту и серверу передать дополнительную информацию.

- Тип запроса (Get, POST, PUT, Delete)

- Тело запроса (во всех запросах кроме Get) - файл с информацией для запроса.

Из чего состоит АПИ (НТТР) ответ?

API ответ состоит:

- Статус кода (1хх, 2хх, 3хх, 4хх, 5хх)

- Тело запроса - файл с информацией для запроса.

Что такое статус коды и какие они бывают?

Статус коды - это коды состояния HTTP/HTTPS.

Они бывают:

1хх - Информационные

2хх - Успех

3хх - Перенаправление

4хх - Ошибка на уровне клиента (клиент не правильно обращается к серверу)

5хх - Ошибка на уровне сервера

Какие методы (типы) НТТП запросов ты знаешь?

POST -метод, который помогает создать определенную сущность.

Get - метод, помогает прочитать, получить информацию об этой сущности.

PUT - метод, помогает изменить сущность.

Delete - метод, удаляет эту сущность.
